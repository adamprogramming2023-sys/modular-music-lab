How it works: A musical scale is an array of semitone intervals from a root note. Selecting a key and scale rebuilds the sequencer rows using those intervals, so every row is guaranteed to sound good with every other row. Semitones are the smallest musical step — like moving one key on a piano.
Module 02 — Step Sequencer 16 steps · 12 notes · click cells to toggle
How it works: Each row is a musical note driven by the Scale Lock module above. Each column is one 16th-note step — 16 steps = one bar of music. The grid stores true/false values in a 2D array. On every tick, active notes in the current column are triggered. Changing the scale rebuilds the row labels automatically.
Module 03 — Synth Engine Waveform · Attack · Release · Volume

Attack 0.02s

How it works: Tone.js creates a PolySynth — plays multiple notes simultaneously. Waveform shape changes the timbre (tone colour). Attack = how fast a note fades IN. Release = how fast it fades OUT. Together these form the ADSR envelope.
Module 04 — Visualiser Live waveform output
How it works: A Tone.js Analyser node taps the audio signal after the synth but before your speakers. Every animation frame (~60fps), JavaScript reads a waveform snapshot and draws it as a line on a canvas element.
